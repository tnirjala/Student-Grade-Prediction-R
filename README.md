# Student Grade Prediction Analysis
R-based data analysis and preprocessing of student performance data.

## Libraries Used
- dplyr, ggplot2, reshape2, readr
- effects, vcd, caret
- FactoMineR, factoextra, tidyr

## How to Run
1. Clone or download this repository
2. Open `student_prediction_analysis.R` in RStudio
3. Go to Session → Set Working Directory → To Source File Location
4. Run the script

## Files
- `student_prediction_analysis.R` — main analysis script
- `Data/student_prediction.csv` — dataset
